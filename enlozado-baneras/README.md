## Estructura
Las landings están como archivos `.html` en la raíz de esta carpeta. No se usan carpetas por landing.

## Stack
HTML estático, Tailwind CDN, CSS propio, JavaScript simple y SVGs inline. No hay imágenes obligatorias.

## SEO
Cada página tiene title, meta description, canonical, Open Graph, schema de negocio/servicio y FAQPage.

## Hosting
Subir los HTML de landings al root público esperado por sus URLs y publicar el index como `/enlozado-baneras` según la configuración del hosting.

## Reglas editoriales
No prometer duración exacta, garantía, tiempos exactos, ni resultados absolutos. Mantener el mensaje de evaluación previa según estado real de la bañera.
