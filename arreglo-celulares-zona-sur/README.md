## Editar textos

Mantener tono claro, técnico y comercial. No prometer precios, tiempos, garantías, reparación en el día ni recuperación de datos si no está confirmado.

## Seguridad

No ofrecer desbloqueos ilegales, bypass de seguridad, cuentas robadas ni eliminación de bloqueos antirrobo.

## Subir a hosting

Subir todos los archivos de esta carpeta al directorio público correspondiente. No requiere build ni dependencias: usa HTML estático, Tailwind CDN, CSS embebido y JavaScript simple.

## Imágenes

Esta versión no usa imágenes ni carpeta `assets/img/`. El diseño visual usa SVG inline, gradientes y patterns CSS.
