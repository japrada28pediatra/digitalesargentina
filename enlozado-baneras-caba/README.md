# Enlozados Capital

Sitio HTML estático para enlozado de bañeras en Capital Federal / CABA.

## Dominio
URL pública configurada: `https://digitalesargentina.com/enlozado-baneras/`.

Para cambiar el dominio, reemplazar en todos los HTML, `sitemap.xml` y `sitemap-index.xml` la base actual por la nueva.

## WhatsApp
Los CTAs usan el placeholder `https://wa.me/5491133161653` y el número visible `+54 9 11 3316-1653`.

Cuando esté confirmado el número real, reemplazar `5491133161653` por el número con código de país y sin signos, y reemplazar `+54 9 11 3316-1653` por el número visible.

## Estructura
Las landings están en la raíz del proyecto como archivos `.html`, no en carpetas.

## Imágenes
Esta versión no usa imágenes ni carpeta de imágenes. El diseño se resuelve con Tailwind CDN, CSS propio, patterns, cards, gradientes y SVGs inline.

## SEO
Cada página tiene title, meta description, canonical, Open Graph, schema LocalBusiness/HomeAndConstructionBusiness y FAQPage.

## Hosting
Subir el contenido de esta carpeta al directorio público correspondiente a `/enlozado-baneras/` y enviar `sitemap.xml` en Search Console.
