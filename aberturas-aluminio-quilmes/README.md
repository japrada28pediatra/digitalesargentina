Web HTML estática para aberturas de aluminio en Quilmes, Bernal y Zona Sur.

## Cambiar dominio
El dominio configurado es:

```text
https://digitalesargentina.com/aberturas-aluminio-quilmes/
```

Actualizar canonical, schema y sitemap si cambia la URL final.

## Reemplazar imágenes
Las imágenes están en:

```text
assets/img/
```

Rutas usadas:

```text
assets/img/aberturas-aluminio-quilmes-hero.webp
assets/img/fabricacion-aberturas-aluminio.webp
assets/img/cerramientos-aluminio-premium.webp
```

Mantener formato `.webp`, imágenes horizontales y sin texto/logos.

## Editar textos
El home está en `index.html`. Las landings están como archivos `.html` en la raíz del proyecto, no en carpetas.

## Subir a hosting
Subir todos los archivos y carpetas de este proyecto:

```text
index.html
*.html
assets/
image-prompts.md
mapa-del-sitio.md
README.md
```

Las landings deben quedar como archivos `.html` en la misma carpeta que `index.html`.

## Recomendaciones SEO
- Mantener un solo H1 por página.
- Usar títulos y meta descriptions únicos.
- Mantener canonical y sitemap con URLs absolutas.
- No cambiar los enlaces internos `.html` por carpetas.
- Mantener los enlaces internos como rutas relativas si la web se publica dentro de una carpeta.
- Mantener CTAs a WhatsApp visibles.
- No prometer precios, tiempos ni garantías no confirmadas.
