# Pilates Burzaco

Web HTML estática para clases de pilates en Burzaco y Zona Sur.

## URL final

```text
https://digitalesargentina.com/clases-pilates-burzaco
```

Las landings se crearon como archivos `.html` dentro de la carpeta del proyecto.

## Cambiar dominio

Buscar y reemplazar:

```text
https://digitalesargentina.com/clases-pilates-burzaco
```

Actualizar también los `canonical`, el schema y `sitemap.xml`.

## Reemplazar imágenes

Las imágenes están en:

```text
assets/img/
```

Mantener nombres y proporción horizontal para no tocar el HTML. Si se reemplazan por versiones optimizadas, conviene usar el mismo nombre de archivo o actualizar los `background-image` de las páginas.

## Subir a hosting

Subir la carpeta completa `clases-pilates-burzaco` al servidor para que la home quede en:

```text
https://digitalesargentina.com/clases-pilates-burzaco
```

Las landings deben quedar como archivos `.html`, por ejemplo:

```text
https://digitalesargentina.com/clases-pilates-burzaco/pilates-burzaco.html
```

No convertir las landings en carpetas.

## SEO

- Mantener un H1 único por página.
- Mantener canonical absoluto por landing.
- Todos los CTAs deben apuntar a WhatsApp.
- No prometer precios, cupos ni fechas si no están confirmados.
- Mantener actualizado `sitemap.xml` si se agregan nuevas landings.
- Enlazar las landings desde home, footer y bloques relacionados.
